## 🌟 Overview

The **Movie Recommendation System** is an intelligent web application that helps users discover movies tailored to their preferences. Using advanced algorithms and an intuitive interface, this system provides personalized movie suggestions based on user ratings, genres, and viewing history.

## ✨ Features

### 🎯 **Smart Recommendations**
- **Personalized Suggestions**: Get movie recommendations based on your viewing history and ratings
- **Genre-Based Filtering**: Discover movies by your favorite genres
- **Similar Movies**: Find movies similar to ones you've enjoyed
- **Trending Movies**: Stay updated with the latest popular films

### 🎨 **User Experience**
- **Responsive Design**: Seamless experience across all devices
- **Intuitive Interface**: Clean and modern user interface
- **Fast Search**: Quick movie search with real-time results
- **Movie Details**: Comprehensive information including cast, plot, and ratings

### 🔧 **Technical Features**
- **Lightweight**: Built with vanilla JavaScript for optimal performance
- **Cross-Platform**: Works on all modern web browsers
- **Docker Support**: Easy deployment with containerization
- **Modular Architecture**: Well-organized codebase for easy maintenance

## 🚀 Quick Start

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- [Docker](https://www.docker.com/) (optional, for containerized deployment)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/jhaabhijeet864/movie_recommendation_system.git
   cd movie_recommendation_system
   ```

2. **Open the application**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # Or use a local server
   python -m http.server 8000
   ```

3. **Using Docker** (Optional)
   ```bash
   # Build the Docker image
   docker build -t movie-recommendation .
   
   # Run the container
   docker run -p 8080:80 movie-recommendation
   ```

## 🎮 Usage

### Getting Started
1. **Launch the Application**: Open `index.html` in your web browser
2. **Search for Movies**: Use the search bar to find specific movies
3. **Rate Movies**: Rate movies you've watched to improve recommendations
4. **Get Recommendations**: Click on "Get Recommendations" to discover new movies
5. **Explore Genres**: Browse movies by different genres

### Features Overview
- **🔍 Search**: Find movies by title, actor, or director
- **⭐ Rating System**: Rate movies from 1-5 stars
- **🎭 Genre Filter**: Filter movies by genre preferences
- **📊 Trending**: Discover what's popular right now
- **💾 Favorites**: Save movies to your favorites list
